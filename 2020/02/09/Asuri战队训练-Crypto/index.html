0x08 Easy-RSA

  1. RSA，已知公钥和密文求明文，使用工具RSAtool；

  2. 图中的 P、Q、R、D、E 分别就是 RSA 算法中的 p、q、N、d、e，右上角选择进制，注意不要弄错，e 只有十六进制可用，所以这里要把 23 换成 17；

  3. 将 N=322831561921859 填入，左下角有一个 Factor N 的按钮，这是分解 N 的意思，点一下，会自动开始分解因数，得到 P=13574881、Q=23781539，再点一下 Calc. D，计算出 d=42108459725927，这时可以看到 Test 按钮不再是灰色，表明可以使用简单的加解密功能，点它，弹出一个框；

  4. 第一个框是明文，第二个框是密文，输入明文 6162，点击 Encrypt，得到密文 178401292768926，这时就可以使用解密功能（必须先用一次加密才行）；

    密文 0xdc2eeeb2782c，换算十进制 242094131279916，点 Decrypt，直接得到字符串 3a5Y。
