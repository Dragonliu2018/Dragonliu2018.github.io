湖湘杯2019复赛--Misc
Created 2020-02-07| Updated 2020-02-08|CTFMisc

something in image

  1. file命令分析，是linux镜像文件；

  2. 使用UltralSO打开镜像文件发现已损坏，再用R-studio打开镜像并恢复所有文件，得到Flag.txt。

Ezmemory

  1. 拿到一个mem.raw，使用volatility获取镜像系统信息；

  2. 列出所有进程，发现cmd.exe(--profile=Win7SP1x64指定操作系统 );

  3. 查看命令行上的操作，发现flag。

miscmisc

  1. 打开后下载附件buguoruci.png，是一个.png后缀的图片，看到图片二话不说直接梭，拖到HXD里面，直接搜索 flag，用F3查找下一处，用winhex分析，我们会看到falg.zip字段，同时也可以看到50 4B 03 04的数字，.zip文件头是50 4B 03 04 ，常见的文件类型和文件头:

    1. 由于发现多个zip格式文件，直接把源文件改成.zip格式，改完后成了一个.zip格式的压缩包，打开压缩包后，有两个文件chadiand.zip和chayidian.jpg；

    2. 打开压缩文件chadian.zip，会看到一个加密的flag.zip文件和一个加密的flag.txt文本; 将chayidian.jpg放入HXD, 搜索 flag ，会看到flag.txt和 .zip文件开头50 4B 03 04，直接把jpg格式改为.zip格式, 发现可以解压，得到一个 flag.txt，刚才解压chayidian.zip文件时，目录下也有一个flag.txt 文件，查看两个文件的CRC(使用Bandizip进行预览，点击查看，添加CRC校验) 可知两个文件一样，所以这是一个明文攻击；

    3. 又已知是.zip加密，使用工具Advanced Zip Password Recover进行解密，获得密码z$^58a4w(AZPR明文使用);

    4. 将加密文件chadian.zip解压，得到3个文件whoami.zip、world.txt、world1.png；打开whoami.zip文件，发现有个加密文本，需要密码，猜想flag就在里面；打开world.doc文件，只有简单几个字除了这个就差一点点了；打开 world1.png图片，发现有提示：pass in world. 此时想到密码可能与 此图片还有world.doc文件有关;

    5. 将world1.png工具 StegSolve中，打开图片然后试探各种通道，在LSB BGR条件下发现pass，所以这是LSB信息隐写，得到pass：z^ea，去解压文件 发现不行(LSB隐写);

    6. 根据提示 pass in world 猜想 world.doc 文件里不可能那么简单, 可能还会有隐藏文字，ctrl+A 全选，右击—字体—取消勾选隐藏。果不其然，发现了隐藏字符;

    7. 最后的密码是 pass内容+world里每行字符串的最后一个字符，就是密码 ：z^ea4zaa3azf8;

    8. 用密码解密得到flag ：flag{12sad7eaf46a84fe9q4fasf48e6q4f6as4f864q9e48f9q4fa6sf6f48}。

elf just elf？

  1. 拿到压缩包，是加密的，Ziperello爆破得密码123456；查看压缩包的注释，发现是 web 端常用的 Quoted-printable 编码，将“%”替换为“=”，解码为注意：这不是逆向，压缩包中密文解密密码已被我隐藏。(Quoted-printable 编码);

  2. 题目为 ELF，而 ELF 可执行文件，将 flag 文件放到 linux 中 file 一下，发现是 32 位的 ELF 文件，可以在 32 位 ELF 文件中隐藏信息的有 hydan(安装参考)，但是hydan解密需要密码。猜测和压缩包密码一样是123456，得到8*&#b；

  3. enc.txt 中的文本有点像 AES 的密文，而注释提示说解开这段密文的密码被隐藏起来了，所以使用密码8*&#b解开这段密文，获得明文 bNa3vN1ImCHYlN42kC1FYA47aMalbAXIpNaMsAXBVNKApMqMnBro8(AES加密解密)

  4. 将解出后的明文进行 XXencode 解码，得fa{d52991da084d0bfblg255ff823aec5cf37}(XXencode 解码);

  5. 最后再栅栏解密即可得到 flag{2d5552f9f9812d3aa0e8c45dc0fb3f7b}(栅栏解密))。
