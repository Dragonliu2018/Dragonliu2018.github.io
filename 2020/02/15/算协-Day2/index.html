0x04 连续自然数和 — 数论

思路

　设连续的自然数的段首、段尾：head, tail；

$\because$　M = $\frac{(head + tail)(tail - head + 1)}{2}$

$\therefore$　2M = (head + tail)(tail - head + 1)

$\therefore$　2M可以分解为两个整数的乘积，设为k1, k2(k1 < k2)

$\therefore$　k1 = (tail - head + 1)　k2 = (head + tail)

$\therefore$　head = $\frac{(k2 - k1 + 1)}{2}$　tail = $\frac{(k1 + k2 - 1)}{2}$

因此：当k1, k2一奇一偶时满足题解，并把head = tail(k1 = 1)情况去除

0x09 公约数的和 — 数论

思路

大牛的题解：

设f[d]=∑∑gcd(i,j)=d

F[d]=∑∑d|gcd(i,j)

不难看出F[d]=n/d*(n/d)

那么f[d]=F[d]-∑f[kd]

所以o(nlogn)扫一下就好了（主要是代码特别短）

然后ans=(∑f[d]-n*(n+1)/2)/2 减去gcd(d,d)=d的和gcd(i,j)=gcd(j,i)重复的
