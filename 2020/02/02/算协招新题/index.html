A. sion的密码

题目描述

　　由于众所周知的原因，然叔需要一套题目来举办比赛。由于可以无限期拖欠薪水，然叔选择了出题性价比极高的sion同学。 为了保密，在出了14道题目之后，sion要将题目加上密码，在比赛开始的那天再交给然叔解封。出于深深的怨念，sion设置的密码是然叔欠他的钱的数额。但sion是一个很讨厌偶数的人，所以他决定将初始的密码中的偶数数位去掉后作为密码交给然叔。 sion的好朋友hansir知道sion讨厌偶数的习惯，并且知道然叔欠钱的数额，所以想要破解sion的密码来达到他不可告人的目的。但由于然叔欠款数额巨大，hansir打算编写一个程序来破译sion的密码，希望你能够帮助他。

输入格式

输入只有一行，为一个正整数n，代表然叔的欠款。

输出格式

输出一个字符串，表示sion所设置的密码。

输入输出样例

输入：1234567890123456　　输出：13579135

说明/提示

对于50%的数据，n是一个小于9位的整数。
对于100%的数据，n是一个小于300位的整数。

思路

　　开一字符数组存放整数n，顺序输出奇数位上的字符即可。

B. sion的整数收集

题目描述

　　sion对完全平方数有很大的兴趣，为了探究完全平方数的性质，sion想收集所有满足以下性质的整数：

  1. 整数可以表示为两个不同的小于等于n的完全平方数的和
  2. 整数小于等于m
    你可以帮助sion统计出满足他要求的整数的数量吗？
    (0也是完全平方数)

输入格式

只有一行，包括两个整数，n，m

输出格式

只有一行，包括一个整数，表示满足sion要求的整数的数量

输入输出样例

输入：9 100　　输出：6

说明/提示

对于100%的数据满足，n<=2000，m<=4000

思路

　　暴力遍历出答案。第一层循环遍历1-m，第二三层判断是否能写成两个不同的小于等于n的完全平方数的和的形式，若满足个数+1。

C. sion玩游戏

题目描述

　　sion最近一段时间沉迷吃鸡，但尝试几百盘仍然没有一次成功吃鸡，sion的好基友hansir实在看不下去sion继续被虐，决定帮sion买个挂。 hansir可以买到的挂分为三种，每种挂有两个参数a，b，a表示这种挂可以让sion的rank分（即游戏中sion的实力）提高a蛇（蛇：为该游戏中的实力计量单位），b表示hansir要购买这种挂需要花费的钱。 虽然hansir是南航acm队有史以来最有钱的男人，但hansir也不想买性价比低的外挂，所以hansir希望你能帮他把三种外挂按照性价比，即a/b的大小从小到大进行排序。

输入格式

包括三行，第i行包含一个字符串，表示第i中挂的卖家名字，然后是两个正整数ai和bi表示外挂的两个参数

输出格式

包含三行，按照性价比从小到大的顺序输出卖家名字，若性价比相同，先输出卖家名字字典序小的，行末没有空格。

输入输出样例

输出 #1复制

思路

　　用结构体存放“挂”，排序即可。

D. sion定理

题目描述

　　sion想要做一个数学家，提出一个以他的名字命名的定理。 今天，他兴奋的发现了一条规律，给sion一个正整数x，不断重复以下操作：

  1. 当x为奇数时，x=x*3+1
  2. 当x为偶数时，x=x/2
    最终会使得x=1
    你能帮助sion演算x从初始到1的全过程吗？

输入格式

仅一行，包括一个正整数x

输出格式

输出操作过程中的每一个x，每个x占一行

输入输出样例

输出 #1复制

思路

　　根据要求模拟即可，好像要开long long。

G. M2akik的素数

题目背景

恭喜你通过了Level1的考验，来到了Makik主持的Level2。

题目描述

　　众所周知Makik一直钟情于素数。有一天Makik决定探究素数的分布。但是Makik是老板，于是他把计算的任务交给了你。 Makik共有n次询问，每次询问给出一个正整数L、R，需要您给告诉李老板在数字[L,R]间，共有多少个素数。

输入格式

第一行一个正整数n
第二行开始共n行，每行两个正整数L，R。
含义见题目描述。

输出格式

对于每个询问输出一行，
每行需给出一个正整数表示数字[L,R]之间的素数个数。

输入输出样例

输入 #1复制

输出 #1复制

说明/提示

对于30%的数据，1≤n≤10, 1≤L≤R≤10w;
对于100%的数据，1≤n≤10w,1≤L≤R≤10w;
输入量过大，请使用scanf进行数据的输入，否则会超时。

思路

　　遍历判断素数个数。

H. Makik的粉丝

题目背景

众所周知Makik有很多粉丝。

题目描述

　　一天Makik在西操开始了算法大讲堂，并且准备了排成一排的L个座位(标号为[1,L])。这L个座位中有n个粉丝坐在某些座位上（设每个座位宽度为1）。 外面突然下起了雨，Makik为了不让他的粉丝们淋雨决定购买防雨棚。由于棚子供应商的支架有限，他只能提供m个防雨棚，但是每个防雨棚可以任意长。Makik为了尽可能的节省成本，他需要使购买的防雨棚总长度和最小。由于Makik是老板，所以他将这个任务交给了你。 Makik会告诉你n个粉丝的位置，需要你告诉他需要购买雨棚的最小总长度。

输入格式

第1行：防雨棚最大的个数m，坐位总数L，和粉丝数n。
第2到n+1行：每行包含一个整数，表示粉丝所占的座位编号。

输出格式

单独的一行一个整数表示所需的防雨棚的最小总长度。

输入输出样例

输入 #1复制

输出 #1复制

说明/提示

对于100%的数据有：1 <= n,m,L <= 100000;

思路

　　逆向思考，用一个防雨棚将所有粉丝罩住，粉丝之间的间隔排序，每减少一个间隔相当于增加一个防雨棚，减少前(m-1)大的间隔即可。

代码

I. Makik的迷弟

题目背景

众所周知Makik有很多迷妹。

题目描述

　　Makik建造了一座有n栋别墅的城堡给自己的迷弟们住。
　　别墅排在一条直线上，第i栋别墅在x i​的位置，但是Makik的m个迷弟彼此争风吃醋，因此经常互相攻击。Makik为了防止迷弟之间互相伤害，因此决定把每位迷弟都放在离其它迷弟尽可能远的别墅。也就是要最大化最近的两个迷弟之间的距离。
　　迷弟们并不喜欢这种布局，而且几个迷弟放在一栋别墅里，男上加男，他们就要发生争斗。为了不让迷弟们互相伤害,Makik 决定让你给他的迷弟们分配隔间，使任意两个迷弟之间的最小距离尽可能的大，那么，这个最大的最小距离是多少呢？

输入格式

第一行用空格分隔的两个整数n和m；

第二行为n个用空格隔开的整数，表示位置x i。

输出格式

输出仅一个整数，表示最大的最小距离值。

输入输出样例

输入 #1复制

输出 #1复制

说明/提示

2≤n≤10e5
0≤x i≤10e9
2≤m≤n

思路

　　最大的最小距离值，二分答案。跳石头)
