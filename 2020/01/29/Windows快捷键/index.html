Windows快捷键
Created 2020-01-29| Updated 2020-01-30|OSWindows

场景

　　2020年1月29日，大年初五，天欲雪。回想之前大师傅们快捷键用得666，菜鸟dragon也想用快捷键，但是记性不好，所以收集起来，以备温习。

常用快捷键

快捷键 功能描述
F2 当你选中一个文件的话，这意味着”重命名”
F11 全屏
Ctrl + C/V 复制/粘贴
Ctrl + X 剪切
Ctrl + Z 撤销
Ctrl + W 在浏览器中直接关闭页面，也可关闭QQ、word等界面
Ctrl + Shift + T 浏览器中恢复之间关闭的页面
Ctrl + Tab 在页面上的各框架中切换（加shift反向）
Alt + Tab 切换当前程序
Shift 输入法切换
Shift + space(空格) 全角切换
Windows + L 直接锁屏
Windows + D 直接显示桌面
Windows + M 收起所有窗口
Windows + E 打开资源管理器查找文件

　注：由于安装一些程序，快捷键可能出现冲突。
