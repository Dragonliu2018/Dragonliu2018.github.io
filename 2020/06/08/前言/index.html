前言
Created 2020-06-08| Updated 2020-07-18|其他

　　时隔四个月，Dragon Liu博客再次上线！！！ 为与之前的博客做区分，特称Blog2.0。
　　构思的时候还感觉有很多话要说，可真要写的时候却无从下手。语文功底着实令人担忧，所以用关键词的方式强行凑起这篇前言。从生态、时间、效率、学习、能力、动力、专业及感悟八方面展开。
　　生态：gitbook + Blog + github + 本地。学期初，时间充沛，受PA讲义的启发使用gitbook编写在线笔记，感觉良好。gitbook没有花里胡哨的页面，很实用，访问速度方面较Blog快；但是没有评论功能，并且框架固定，十分死板。再者说，这个Hexo+github构建的Blog花费了我大量时间，放弃他那就太亏了。经过一番思考，决定使用这样的搭配，他们是一个整体，相互联系，各有侧重。gitbook：侧重理论知识，由大学课程、阅读笔记、机器学习(放在这里是最好的安排😁)构成；Blog：侧重动手实践+课外拓展，主要为CTF设计，由Pwn、Web、Reverse、Crypto、Misc、Develop、Algorithm、其他构成；github：主要贮存机器学习重要代码、资料；本地：负责副本。
　　时间：时间是让人猝不及防的东西。前天是2020年6月6日，距离2018年6月6日已有2年的时间，距离2019年的6月6日已有1年的时间。高考似乎仍在眼前，对爱情的试探仍无法忘记。但是时间不会管你发生了什么，他只管做好自己的本职工作。作为菜鸟，应该什么事情都早有打算，早开始着手做。像一些作业啥的，不应让他赶着自己，而是自己赶着他，然后和同学、助教、老师讨论。
　　效率：与时间的关系是剪不断、理还乱。效率是我一直渴望的，但是喜欢拖延，学习的同时娱乐，这让我和效率显得格格不入。一次一次地整理好书桌、擦拭好笔记本和键盘、洗个热水澡、舒舒服服睡一觉，明天迎来一个崭新的开始，但是明日复明日，明日何其多。大学前的暑假就发觉短视频太浪费时间，一旦点开，手指就停不下来，为此特意删除抖音app；但是又栽在了QQ看点(还没到要删除QQ的地步😅)。
　　学习：学习是第一生产力。一方面，三人行，则必有我师。学习+mu的活学活用，学习Ocean的人际交往，学习Forest的办事能力…另一方面是课程学习，现在养成了一个极其恶劣的习惯，上课跟不上节奏，想着最后考试的时候突击；课后作业没有自己的思考，先抄答案后看懂就满足，PA面向github编程等等。再一方面，学习新知识静不下心来，就像CTF和计算机视觉还没入门。不过，这个暑假打算突击一下数模，找回失踪若干年的数学感觉(仅仅是个美好的期望)。这个生态很大程度上为了解决学习这一问题而创建的。

　　能力：能力是直接影响效率的。在大一时，听讲座、打比赛、做科创，啥也不懂、等待”躺赢“会以低年级学生理应如此进行敷衍；大二的时候仍然可以用这个理由搪塞，但有些脸红；倘若大三仍然在用这个理由逃避就有些。。。😭
　　动力：若没有人赠予你阳光，那就做自己的小太阳。对于我来说，动力若隐若现、若即若离、恍兮惚兮。没有动力，就努力创造心之所向吧😋。Blog1.0更多的是为了写博客而写博客，2.0要求体现自己的思考。
　　专业：努力提高专业素养。本学期有较多的常识课，不管是信息安全导论，还是信息安全法律与规范，通过上课回答问题的情况可以看出：我与同学们的差距很大。自己一点儿专业素养都没有，而且不关注信息安全的动态。
　　感悟：一切都是最好的安排。我是十分赞成这句话的，不过我对这里的安排有这样的理解：这种安排是万事俱备后，等待有把握的或意想不到的安排；而不是呆在原地、拜天拜地去祈求好的安排。
　　综上所述，希望Blog2.0可以构建一个良好的学习生态环境。
