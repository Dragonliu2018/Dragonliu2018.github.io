Manjaro Linux安装以及日常配置
Created 2019-10-09| Updated 2020-02-01|Linux

目的

不满足于当前windows,想要安装Linux

操作

  • 镜像下载

    1. 官网下载
      Ubuntu Kali
    2. 清华源下载(速度较快)
      清华源

      补充->高速下载：FTP客户端下载,例如Filezilla。

    3. 哈希值文件校验

  • 分盘

  • 烧录
    1. 软碟通工具(UltralSO)，但是要切记使用raw刻录方式。
    2. Linux命令行：
      复制： dd if=/dev/target of=/tmp/source.img
      显示进度：watch -n 5 killall -USR1 dd

  • 安装

    • 虚拟机安装
      使用Vmware或VirtualBox
    • 物理机安装
    1. 关闭掉Secure Boot
    2. 分区(重要)
      (1) mbr启动模式
      这里我们直接划分出来一块分区文件系统选择ext4，挂载点 /

    (2)uefi启动模式
    这里需要一个esp分区
    划出300MB分区，文件系统fat16，挂载点boot/efi，标记boot，esp
    还需要一个根分区也就是系统分区
    划出分区，文件系统ext4，挂载点 / ，无需标记

  • 进行软件源设置以及系统更新

    1. 换源

      • 弹出窗口选择中科大

      • 粘贴以下内容

        sudo pacman -Syy && sudo pacman -S archlinuxcn-keyring

    • 系统更新（出现替换提示选择Y）

  • 中文输入法
