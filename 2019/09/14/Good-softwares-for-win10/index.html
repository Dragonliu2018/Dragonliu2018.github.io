Good softwares for win10
Created 2019-09-14| Updated 2019-09-14|Blog

文件搜索增强工具: Everything && Wox

Everything本身很小，搜索速度很快，甚至可以删除所有桌面应用快捷方式。

Wox可以调用 Everything 快速搜索本地文件，也可以启动程序、执行命令行、进行网页搜索等，非常强大。Wox 支持安装各种插件，从而实现剪贴板历史、进程管理、文本翻译、进制转换、快递查询等扩展功能。

注：显示wox安装失败时，alt+空格，若显示wox搜索框，则表明已安装成功。

查看增强工具: Seer

安装 Seer 后，无需其它设置，你就可以在选中文件后按空格快速预览它了。Seer 默认支持文本、图片、视频、压缩包等类别的上百种文件扩展名，你也可以在设置中为其安装 Office、CAD 等扩展插件，让其支持更多文件类型。此外，Seer 还支持与 Directory Opus 等第三方程序集成，或者自定义预览快捷键

粘贴板增强工具: Ditto

无论你粘贴了什么，任何时候都已回溯，支持导出，修改，搜索。

截图增强工具：Snipaste

截图，贴补，保存图，小巧，功能丰富。

markdown写作工具：typora

win平台颜值最高的markdown写作工具

压缩解压缩软件：Bandizip

没有广告，软件体积小，响应快，丰富右键菜单，从不打扰用户的免费压缩软件。

视频播放软件： PotPlayer

avi格式都支持，支持倍速播放，可以看世姐任意一个国家的电视，免费，无广告。

浏览器：谷歌，火狐

编辑器： Visual Studio Code
