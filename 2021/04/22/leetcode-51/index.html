leetcode-51 N皇后 (困难)
发表于 2021-04-22| 更新于 2021-04-22|算法

题目

n 皇后问题 研究的是如何将 n 个皇后放置在 n×n 的棋盘上，并且使皇后彼此之间不能相互攻击。

给你一个整数 n ，返回所有不同的 n 皇后问题 的解决方案。

每一种解法包含一个不同的 n 皇后问题 的棋子放置方案，该方案中 ‘Q’ 和 ‘.’ 分别代表了皇后和空位。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/n-queens
著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。

思路

思路1 基于集合的回溯

dfs每行，然后遍历每列。为了判断一个位置所在的列和两条斜线上是否已经有皇后，使用三个集合(set)分别记录每一列以及两个方向的每条斜线上是否有皇后。

注意当前答案状态的回溯。

  • 时间复杂度：$O(n!)$
  • 空间复杂度：$O(N)$，其中 N 是皇后数量。空间复杂度主要取决于递归调用层数、记录每行放置的皇后的列下标的数组以及三个集合，递归调用层数不会超过 N，数组的长度为 N，每个集合的元素个数都不会超过 N

思路2 基于位运算的回溯

  • 时间复杂度：$O(n!)$
  • 空间复杂度：$O(n)$
