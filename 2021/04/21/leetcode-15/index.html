leetcode-15 三数之和 (中等)
发表于 2021-04-21| 更新于 2021-04-21|算法
字数总计: 746|阅读时长: 3 分钟|阅读量:

题目

给你一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，使得 a + b + c = 0 ？请你找出所有和为 0 且不重复的三元组。

注意：答案中不可以包含重复的三元组。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/3sum
著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。

思路

* 思路1 哈希表

与leetcode-1 两数之和 (简单)思路相似，两次遍历，然后借助map查找-a-b是否存在。

仍存在重复答案的问题未解决：[0,0,0,0]

  • 时间复杂度：$O(n^2)$
  • 空间复杂度：$O(n)$

思路2 双指针

先把数组排序，遍历数组，双指针分别指向当前元素后一个及最后一个。按规则移动双指针：

  1. a+b+c<0：b++
  2. a+b+c=0：获得一个答案
  3. a+b+c>0：c—

直到双指针相遇。

注意：判断是否有重复答案，也就是只要在遍历过程中保证a、b不重复即可。

  • 时间复杂度：$O(n^2)$
  • 空间复杂度：：$O(\log N)$。我们忽略存储答案的空间，额外的排序的空间复杂度为 $O(\log N)$。
