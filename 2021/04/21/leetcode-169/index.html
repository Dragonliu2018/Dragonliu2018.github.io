leetcode-169 多数元素 (简单)
发表于 2021-04-21| 更新于 2021-04-21|算法
字数总计: 672|阅读时长: 2 分钟|阅读量:

题目

给定一个大小为 n 的数组，找到其中的多数元素。多数元素是指在数组中出现次数 大于 ⌊ n/2 ⌋ 的元素。

你可以假设数组是非空的，并且给定的数组总是存在多数元素

进阶：尝试设计时间复杂度为 O(n)、空间复杂度为 O(1) 的算法解决此问题。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/majority-element
著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。

思路

思路1 map

key是元素，value是出现的次数。

  • 时间复杂度：$O(n)$
  • 空间复杂度：$O(n)$

思路2 sort

排下序，取中间值即可。

  • 时间复杂度：$O(nlogn)$
  • 空间复杂度：$O(logn)$，算排序时花费的空间。

思路3 分治

将原数组一分为二，分别解决。

  • 时间复杂度：$O(nlogn)$
  • 空间复杂度：$O(logn)$

思路4 随机算法

因为超过 n/2 的数组下标被众数占据了，这样我们随机挑选一个下标对应的元素并验证，有很大的概率能找到众数。由于一个给定的下标对应的数字很有可能是众数，我们随机挑选一个下标，检查它是否是众数，如果是就返回，否则继续随机挑选。

  • 时间复杂度：期望的时间复杂度为 $O(n)$
  • 空间复杂度：$O(1)$。
