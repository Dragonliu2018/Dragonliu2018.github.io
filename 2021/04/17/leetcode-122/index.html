leetcode-122 买卖股票的最佳时机 II (简单)
发表于 2021-04-17| 更新于 2021-04-17|算法
字数总计: 539|阅读时长: 2 分钟|阅读量:

题目

给定一个数组 prices ，其中 prices[i] 是一支给定股票第 i 天的价格。

设计一个算法来计算你所能获取的最大利润。你可以尽可能地完成更多的交易（多次买卖一支股票）。

注意：你不能同时参与多笔交易（你必须在再次购买前出售掉之前的股票）。

链接：https://leetcode-cn.com/problems/best-time-to-buy-and-sell-stock-ii

思路

思路1 贪心

  1. 每次选择贡献大于 0 的相邻区间即能使得答案最大化，因此最后答案为$\textit{ans}=\sum_{i=1}^{n-1}\max{0,a[i]-a[i-1]}$
  2. 注意：贪心算法只能用于计算最大利润，计算的过程并不是实际的交易过程
  • 时间复杂度： $O(n)$
  • 空间复杂度：$O(1)$

思路2 dp

leetcode-188 买卖股票的最佳时机 IV中k=n的特例，直接套板子会使得dp数组过大，所以降维。

  1. 状态：$dp[i][j]$ 代表第$i$天有$j$个股票在手的最大利润，其中$0<=i<n$代表第几天、$j=0或1$代表当天是否有股票在手
  2. 状态转移方程：
    • $dp[i][0] = max { dp[i-1][0],dp[i-1][1]+prices[i] }$，对应情况不动、卖出
    • $dp[i][1] = max{dp[i-1][1],dp[i-1][0]-prices[i]}$，对应情况不动、买入
  3. 答案：$dp[n-1][0]$
  4. 注意：初始化及边界条件、由于买入与抛出交易数都会增加，所以交易笔数*2。
  • 时间复杂度：$O(n)$
  • 空间复杂度：$O(n)$
