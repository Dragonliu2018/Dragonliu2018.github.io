0x01 你好，i春秋(10pt)

　　根据题目关注i春秋微信（微信号：icqedu）回复CTF即可。关注微信号后，输入CTF，并没给出flag，而是给了CTF实战技巧文章推荐，这时还需要给出一些关键词，跟机器人问答一点都不流畅~

题目出自：2016全国大学生信息安全竞赛

0x02 Hello World!(10pt)

　　In this capture the flag competiton you’re hunting for these strings, we call them “flags”. These flags always begin with “IceCTF{“ followed by a message in 1337sp34k and end with “}”. Here’s an example flag “IceCTF{h3l10_wr0ld}”. Try submitting it through the text box down below!

题目出自：IceCTF

0x03 那些年我追过的贝丝(10pt)

  • 题目提示贝丝贝丝，我爱你（大声循环2的6次方ing），可得base64加密。
  • Base64解密：http://ctf.ssleye.com/base64.html

题目出自：“百度杯”CTF比赛 十月场

0x04 签到题(10pt)

  • 题目内容：对百度杯用时下最流行的表白（去调戏i春秋公众号）大声说出你的爱！！！
  • 在i春秋官微输入百度杯么么哒即可获得flag

题目出自：“百度杯”CTF比赛 十月场

0x05 我要变成一只程序猿(10pt)

  • 题目给出一个txt文件，打开发现是用c语言写得一段程序；
  • 将文件后缀改为cpp，在dev-cpp中运行；
  • 若报错，将void main改为int main
  • 输入给定字符串得到另外一个字符串，该字符串为flag

题目出自：“百度杯”CTF比赛 十月场

0x06 Vape Nation(10pt)

  • 题目提示Go Green!，应该是简单的图片隐写题；
  • 使用Stegsolve.jar打开图片，点到green plane 0 模式即可获得flag。

题目出自：IceCTF

0x08 challenge(10pt)

  • 所给字符不超过f，即认为十六进制，将其每两位转成ASCII值，即可得flag
  • 16进制到文本字符串：https://www.bejson.com/convert/ox2str/

题目出自：“百度杯”CTF比赛 十月场

0x10 小可爱(10pt)

  • 题目给出一照片，提示txt文件内容即为密码，猜测照片中还包含一个txt文件；
  • 执行命令binwalk test.jpg，果然发现里面还包含其它文件；
  • 执行命令foremost test.jpg，得到压zip缩包，根据输出提示密码是八位数字；
  • 使用Ziperello.exe暴力破解压缩包，得到一张图片，是破碎的二维码，需要拼凑起来(能打印出来就好了)，然后扫码得flag

题目出自：“百度杯”CTF比赛 十一月场
