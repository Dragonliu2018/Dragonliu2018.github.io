201312-4 有趣的数(数位DP)

解题

采用数位DP的思想，将所有情况分为6种状态：

情况0：只出现 2(22…)
情况1：只出现 2 0(0、2无顺序要求)
情况2：只出现 2 3(22…33…)
情况3：只出现 2 0 1(00…11…、2位置任意)
情况4：只出现 2 0 3(22…33…、0位置任意)
情况5：只出现 2 0 1 3(22…33…、00…11…、可交叉)

对于以上六种情况，我们最后的求解应该是第六种情况，递推关系式如下：

//sum[i][j]表示第i位数填充后j类型状态的个数
0: sum[i][0] = 1 //只有一种情况222…
1: sum[i][1] = sum[i-1][0]+sum[i-1][1]2 //情况0填充0，情况1填0或2(0、2无顺序有要求)
2: sum[i][2] = sum[i-1][0]+sum[i-1][2] //情况0填充3，情况2填3(2在3之前)
3: sum[i][3] = sum[i-1][1]+sum[i-1][3]2 //情况1填充1，情况3填2或1
4: sum[i][4] = sum[i-1][1]+sum[i-1][2]+sum[i-1][4]2 //情况1填充3，情况2填0，情况4填0或3
5: sum[i][5] = sum[i-1][3]+sum[i-1][4]+sum[i-1][5]2 //情况3填充3，情况4填充1，情况5填3或1

输出结果：sum[n][5]。

注意：sum状态数组需要long long类型，避免中间计算过程溢出。

201403-4 无线网络(BFS)

解题

  1. 构建无向图，路由器为图的点，两个路由器的距离小于r则存在边(增设路由器也全部加入图中，避免m选k以降低复杂度，后面会做合法性检验)；
  2. 求解的是最短路径，图所有边的权重均为1，因此可以用广度优先搜索解决；
  3. 搜索的时候需要记录路径上所经过的增设路由器的个数，并确保不会超过k(合法性检验)；
  4. 使用bfs的层次遍历，可以方便地得到第1个路由器与第2个路由器之间的距离。
  • 参考
    • 图的深搜与广搜
    • CCF计算机职业资格认证考试题解

201409-4 最优配餐(BFS)

解题

  1. 单边长度为1的最短路径求解，可以使用广度优先搜索解决
  2. 不同于普通的广度优先搜索，初始队列的位置不是一个点而是多个点，也就是从多个店的位置同时开始广度优先搜索；
  3. 不能经过的点将其设为visited，方便判断；
  4. 可以把方格中每一个点到最近商店的距离保存到矩阵里，最后再遍历客户计算成本；
  5. 需要注意的是，结果会超过int的表示范围，使用long long类型。
  • 参考
    • CCF CSP 201409-4 最优配餐

201412-4 最优灌溉(Prim)

解题

  1. 建立加权无向图，最小生成树算法解决；
  2. 图用邻接矩阵表示，图的边可能有多次输入，应当取最小值；最小生成树算法用Prim
